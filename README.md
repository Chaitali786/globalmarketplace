"# globalmarketplace" 
This marketplace uses Dynamic Routing and URL Queries to filter products. You can manually type these paths into your browser's address bar to test the functionality:

1. View Category Pages
To see all products in a specific category, use these paths:

Clothing: http://127.0.0.1:3035/cloths

Electronics: http://127.0.0.1:3035/electronics

2. Search by Brand (Query Parameters)
To filter the results inside a category by a specific brand, add the ?brand= query to the end of the URL:

Search for Levis Clothing: http://127.0.0.1:3035/cloths?brand=Levis

Search for Philips Electronics: http://127.0.0.1:3035/electronics?brand=Philips
